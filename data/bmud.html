How to make a server list:
edit ~/.bmud/mudservers
each mud is on its own line. The format is a hostname or ip, followed by a space, then followed by the port number.

mud1.mud.com 4000
127.0.0.1 5333


How do I disable the status box?
go into the preference dialog (file->prefs) and choose "misc". Then set the stat box size to 0. Any text that would have been sent to the stat box will now be sent to the main window
